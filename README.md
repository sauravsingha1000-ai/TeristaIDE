## ✨ About

**TeristaIDE** is a customized and enhanced version of AndroidIDE, designed for advanced Android development and experimentation directly on mobile devices.

It aims to provide a powerful, flexible, and developer-focused environment similar to desktop IDEs.

---

## 🔥 Features

- [x] Gradle support  
- [x] `JDK 11` and `JDK 17` support  
- [x] Built-in terminal  
- [x] Custom environment variables  
- [x] SDK tools access (via terminal)  
- [x] Logcat viewer  
- [x] Git support  
- [x] Java & XML support  
- [ ] Kotlin support (in progress)  
- [ ] UI Designer (partial)  
- [ ] Asset tools (planned)  

---

## 📦 Installation

> ⚠️ This project is under modification and may be unstable.

- Clone the repository  
- Build using Android Studio or GitHub Actions  
- Install APK on your Android device  

---

## ⚠️ Limitations

- Requires Android Gradle Plugin 7.2+  
- Some SDK/NDK tools may not work fully on Android  
- Still under development
